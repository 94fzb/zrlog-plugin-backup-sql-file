# backup-sql-file

> zrlog 备份插件，支持定时备份，实时导出，备份文件加密

### 支持的系统列表

- mac (intel & aarch64)
- linux (amd64 & aarch64)
- windows (intel)

### dev

```shell
export JAVA_HOME=${HOME}/dev/graalvm-jdk-latest
export PATH=${JAVA_HOME}/bin:$PATH
```
